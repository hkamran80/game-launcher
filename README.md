# Game Launcher
Launch all your games, from different launchers, from one page

## Features
* Launchers
  * Supported
    * Steam
    * Origin
  * Planned
    * Epic Games
* Poster Downloads
  * Direct from Steam (for Steam games)
  * From IGDB (for all other launchers)
